MP3 - самый популярный формат, совместим со всеми устройствами.

ПАРАМЕТРЫ:

ЧАСТОТА ДИСКРЕТИЗАЦИИ (Hz)
• 48000 - стандарт для видео
• 44100 - стандарт для музыки (CD качество)
• 32000 - для речи, радио
• Ниже - ухудшение качества

БИТРЕЙТ (кбит/с)
• 320 - максимальное качество, большой файл
• 256 - отличное качество
• 192 - хорошее качество (рекомендуется)
• 128 - среднее качество, компактно
• 96 и ниже - для речи, подкастов

ТИП БИТРЕЙТА
• CBR - постоянный битрейт (предсказуемый размер)
• ABR - средний битрейт (адаптивный)
• VBR - переменный битрейт (лучшее качество/размер)

РЕЖИМ КАНАЛОВ
• Stereo - полноценное стерео
• Joint Stereo - оптимизированное стерео
• Mono - моно (для речи)

КАЧЕСТВО/СКОРОСТЬ
• 0 - максимальное качество (медленнее)
• 9 - максимальная скорость (хуже качество)

СОВЕТ: Для большинства пользователей подойдут:
• 44100 Hz, 192 kbps, VBR, Stereo, качество 2"

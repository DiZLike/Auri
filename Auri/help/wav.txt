WAV - формат без сжатия (PCM).
Максимальное качество, очень большой размер.

ПАРАМЕТРЫ:

ЧАСТОТА ДИСКРЕТИЗАЦИИ (Hz)
• 48000 - стандарт для видео
• 44100 - стандарт для музыки (CD качество)
• 32000 - для речи, радио
• Ниже - ухудшение качества, меньший размер

РАЗРЯДНОСТЬ (бит)
• 16 - стандарт CD (достаточно для большинства)
• 24 - студийное качество (динамический диапазон)
• 32 - для профессиональной обработки

РЕЖИМ КАНАЛОВ
• Stereo - стерео
• Mono - моно

ПРИМЕРНЫЙ РАЗМЕР (для 1 минуты стерео):
• 44100/16: ~10 МБ/мин
• 48000/24: ~17 МБ/мин

СОВЕТ: Для CD качества: 44100/16
Для студийной работы: 48000/24 или выше

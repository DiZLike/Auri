Opus - современный формат с лучшим сжатием.
Качество выше MP3 при том же размере.

ПАРАМЕТРЫ:

ЧАСТОТА ДИСКРЕТИЗАЦИИ (Hz)
• Только 48000

БИТРЕЙТ (кбит/с)
• 192 - отличное качество (прозрачно)
• 128 - высокое качество
• 96 - хорошее качество
• 64 - приемлемо для фона
• 32 - для речи

ТИП БИТРЕЙТА
• VBR - переменный битрейт (рекомендуется)
• CVBR - ограниченный переменный
• HARD-CBR - постоянный битрейт

РАЗМЕР КАДРА (мс)
• 20 - стандарт (рекомендуется)
• 40-60 - для потокового вещания
• 10-2.5 - минимальная задержка

ОПТИМИЗИРОВАТЬ ДЛЯ
• Music - для музыки
• Speech - для речи (лучше передает голос)

РЕЖИМ КАНАЛОВ
• Stereo - стерео
• Mono - моно

СКОРОСТЬ/КАЧЕСТВО
• 0 - максимальная скорость
• 10 - максимальное качество

СОВЕТ: Для музыки: 128 kbps, VBR, 20 ms, Music, качество 10

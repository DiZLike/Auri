# Auri — аудиоконвертер с душой

**Преобразуйте музыку с умом. Мощь кодеков. Простота интерфейса.**

Auri — современный инструмент для преобразования аудиофайлов, созданный с вниманием к деталям и удобству пользователя. Программа объединяет мощь профессиональных кодеков и интуитивно понятный интерфейс, позволяя как быстро сконвертировать пару треков, так и настроить параметры кодирования с точностью до битрейта и размера кадра.

---

## 🎵 Поддерживаемые форматы

### Входные форматы (полный спектр)
WAV, MP3, Opus, AAC, M4A, MP4, AC3, Musepack (MPC), Speex (SPX), TTA, APE, FLAC, WMA, WavPack (WV), WebM, ALAC.

### Выходные форматы

| Формат | Тип сжатия | Особенности |
|--------|------------|-------------|
| **MP3** | С потерями | Максимальная совместимость, работает везде |
| **Opus** | С потерями | Лучшее качество при низком битрейте, современный стандарт |
| **QAAC** | С потерями | Высокое качество, формат Apple (M4A) |
| **FLAC** | Без потерь | Идеально для архивирования и аудиофилов |
| **WAV** | Без сжатия | Студийное качество, максимальная точность |

---

## ⚙️ Тонкая настройка

Для каждого формата доступны как готовые пресеты (низкое, среднее, высокое, максимальное качество), так и **ручной режим** для опытных пользователей.

### Доступные параметры регулировки:
- 🎚️ Тип битрейта (CBR, ABR, VBR, CVBR)
- 🎛️ Частота дискретизации
- 🔊 Режим каналов (стерео/моно)
- 🗜️ Уровень сжатия (для FLAC)
- ⏱️ Размер кадра (для Opus, влияет на задержку)
- 📦 Параметры LossyWAV для FLAC

> **✨ Режим эксперта** — активируется при выборе «Пользовательское» качество. Открывает доступ к продвинутым параметрам кодирования: предустановки CBR/VBR, сложность кодирования Opus, настройки VBR для QAAC и многое другое.

## 🧠 Интеллектуальная работа с файлами

Auri не просто конвертирует — она понимает музыку и заботится о вашей коллекции:

- ✅ Сохраняет все теги (исполнитель, альбом, название трека, номер трека, год, жанр, обложка)
- ✅ Создаёт структуру папок по вашему шаблону: `[artist]\[album]\[track]. [title].[format]`
- ✅ Показывает подробную информацию о каждом файле: битрейт, частота, канальность, кодек, длительность
- ✅ Подсвечивает новые добавленные файлы, предотвращает дублирование в списке

## 👌 Удобство для новичков. Мастер быстрого старта

При первом запуске вас встретит **мастер быстрого старта**. Ответив на несколько простых вопросов:

- Где вы слушаете музыку? (в наушниках, в машине, на акустике)
- Что для вас важнее: качество или размер файла?
- Есть ли особые требования к сжатию?

Вы получите оптимальные настройки для ваших задач. В дальнейшем кнопка **«Быстрая конвертация»** позволит обрабатывать файлы в один клик, используя рекомендованные параметры.

> 💡 **Совет:** Мастер можно запустить повторно через меню «Помощь» → «Мастер настройки».

|--------|------------|-------------|----------------|
| **MP3** | С потерями | Универсальный, работает на любом устройстве | Автомагнитолы, старые плееры, максимальная совместимость |

## 🔍 Продвинутые технологии

### 📉 LossyWAV для FLAC

LossyWAV — это препроцессор, который удаляет незначительные детали звука, неразличимые для человеческого уха, позволяя FLAC сильнее сжать файл. Результат: файл на **30–50% меньше** при практически незаметной разнице в качестве. Идеально для портативных устройств и экономии места в коллекции.

> 🧪 **Технический факт:** LossyWAV не создаёт отдельный формат, он лишь подготавливает аудиопоток, после чего FLAC кодирует его как обычный FLAC-файл. Совместимость со всеми плеерами сохраняется.

### ⏱️ Размер кадра в Opus

Opus позволяет выбирать размер кадра от 2.5 до 60 мс:
- **Маленький кадр (2.5–10 мс)** — минимальная задержка, идеально для видеозвонков и игр
- **Большой кадр (20–60 мс)** — повышает эффективность сжатия на низких битрейтах, делая звук чище

В Auri вы можете настроить этот параметр вручную в режиме эксперта.
